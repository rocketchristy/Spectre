NXTCG v 0.0.1 Release Notes: What’s New and What’s Fixed

Overview

This release of the NXTCG platform introduces core marketplace functionality, allowing users to buy, sell, and manage NextGen trading cards through a fully integrated frontend and backend experience. The release focuses on establishing the end-to-end purchase flow, product configuration, user profile management, and platform stability.

What’s New

NXTCG now supports a complete trading card marketplace experience. Users can browse available cards, configure product attributes, add customized items to their cart, and securely complete checkout. Sellers can list cards for sale and manage their inventory directly from their user profile. The platform also introduces a modern storefront interface, navigation components, and landing pages to support discovery and usability.
